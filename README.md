# e-plantShopping

## Paradise Nursery Shopping App

This repository, **e-plantShopping**, contains a React-based shopping cart application for an online plant shop called Paradise Nursery.

Users can:
- Browse different categories of house plants
- Add plants to a shopping cart
- Manage item quantities
- View the total cost dynamically

The app uses:
- React functional components
- React Router for navigation
- Redux Toolkit for state management
- Dynamic rendering of plant data
